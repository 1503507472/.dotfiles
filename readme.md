### install

```zsh
# dotfiles
git clone --recursive https://github.com/shuvalov-anton/.dotfiles && ~/.dotfiles/init 

# vim's things
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

# macbook's name
scutil --set HostName a8h333
```
